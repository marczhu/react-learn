离线转换

安装命令行工具 npm install -g react-tools
执行 jsx --watch src/ build/
执行完毕后会生成 build/helloworld.js，内容如下：

React.render( React.createElement('h1', null, 'Hello, world!'), document.getElementById('example') );

But在浏览器中仍然无法正常显示 Hello,World!

webpack转换

npm install --save react react-dom babelify babel-preset-react babel-preset-es2015

npm install babel-core babel-loader

webpack ./src/helloworld.js ./build/helloworld.js --module-bind 'js=babel-loader'

在window7下执行 webpack，但是在MacOS下可以正确执行，并且在MacOS可以正常显示!!!

webpack.config.js转换

module.exports = { entry: './src/helloworld.js', output: { path: "build", filename: 'helloworld.js' }, resolve: { extensions: ['', '.js', '.jsx'] }, module: { loaders: [ { test: /\.js$/, loaders:['babel']} ] } }

直接执行webpack即可
